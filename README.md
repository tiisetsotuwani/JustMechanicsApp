
# On-Demand Mechanic App

This is a code bundle for On-Demand Mechanic App. The original project is available at https://www.figma.com/design/Re6C08Wo7fiaTQWnGY6tX0/On-Demand-Mechanic-App.

## Running the code

Install dependencies:

```bash
corepack pnpm install
```

Start the Vite development server:

```bash
corepack pnpm dev:preview
```

Open the app at `http://localhost:5173/`.

Do not use Live Server or `127.0.0.1:5500` for this project. It is a Vite React app and must be run through Vite.

## VS Code

This workspace now includes:

- `.vscode/tasks.json` with `JustMechanic: Start Dev Server`
- `.vscode/launch.json` with `JustMechanic: Open App`

If you use VS Code, you can start the app from `Run and Debug` without manually typing the command.
